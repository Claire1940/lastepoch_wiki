## 現代のビルドプランナーの主要機能

高品質な**Last Epoch ビルドプランナー**は、単にスキルツリーを表示するだけではありません。それは本格的なキャラクターシミュレーターとして機能します。現在の2026年のメタにおいて、本格的な理論構築には以下のいくつかの機能が不可欠とされています。

### スキルおよびパッシブツリーのシミュレーション
Last Epochのすべてのクラスは、基本クラスツリーと3つのマスタリーツリーにアクセスできます。プランナーではポイントを割り当て、ノードがどのように相互作用するかを正確に確認できます。例えば、あるノードが「知力1ポイントにつきダメージ増加」を付与する場合、プランナーはシミュレーター内で装備しているギアに基づいて、実際のダメージ出力を計算します。

### ウィーバーツリー (The Weaver Tree)
最近のアップデートで導入された「ウィーバーツリー」は、さらなるカスタマイズ層を追加しました。現在のプランナーにはウィーバーツリー進行専用のタブが含まれており、運要素の強い「ウィーバーの意志（Weaver’s Will）」アイテムのアップグレードをシミュレートできます。これは、特定のユニークアイテムが高ティアの崇高（Exalted）アイテムよりも投資する価値があるかどうかを判断する上で非常に重要です。

### 装備とアイドルの管理
装備タブは、ほとんどの「計算」が行われる場所です。任意のベースアイテムを選択し、プレフィックス（接頭辞）とサフィックス（接尾辞）を適用し、さらにアフィックスのティアを調整することもできます。

| 装備スロット | 2026年における重要な考慮事項 |
| :--- | :--- |
| **兜 / 胴** | クラス固有のスキルレベルと生命力（Vitality）に焦点を当てる。 |
| **武器** | 固定ダメージと攻撃/キャスト速度を優先する。 |

## キャラクターをインポートする方法

**Last Epoch ビルドプランナー**の最も強力な側面の一つは、ライブキャラクターをインポートして、どこを改善できるかを確認できる機能です。オンラインでプレイしているかオフラインかで、プロセスは若干異なります。

### オンラインキャラクターのインポート
1. プランナーを開き、**Import（インポート）**ボタンを選択します。
2. アカウント名を入力します（ゲーム内のソーシャルメニューで確認できます）。
3. リストから特定のキャラクターを選択します。
4. プランナーが自動的にスキル、パッシブ、現在装備しているギアを反映させます。

### オフラインキャラクターのインポート
オフラインプレイヤーの場合、キャラクターのセーブファイルを見つける必要があります。2026年現在、これらは通常以下のディレクトリにあります。

*   **Windows:** `C:/Users/[ユーザー名]/AppData/LocalLow/Eleventh Hour Games/Last Epoch/Saves`
*   **Steam Deck/Linux:** `[Steamフォルダ]/steamapps/userdata/[ID]/899770/ac/`

ファイル名は通常 `1CHARACTERSLOT_BETA_#` のようになっています。このファイルを**Last Epoch ビルドプランナー**にアップロードすると、ハードドライブにあるキャラクターの状態が正確に再現されます。

## Shattered Omens（シーズン4）に向けたステータスの最適化

シーズン4では新しい状態異常や環境ハザードが導入され、防御のレイヤリング（層状化）がこれまで以上に重要になっています。**Last Epoch ビルドプランナー**を使用する際は、攻撃に完全に集中する前に、防御ステータスの特定の「適正範囲」を目指すべきです。

> **⚠️ 警告:** 物理耐性を無視しないでください。ヴォイド（Void）やネクロティック（Necrotic）ダメージは派手ですが、「Shattered Omens」シーズンにおける「一撃死（ワンショット）」メカニクスの大部分は、依然として強力な物理スケーリングに依存しています。

### 防御ステータスの優先順位
1.  **耐性（Resistances）:** すべてのタイプで75%のキャップを目指しましょう。Last Epochでは、高レベルエリアで耐性がキャップを下回っていると、耐性がマイナスになる可能性があることを忘れないでください。
2.  **クリティカル回避（Critical Strike Avoidance）:** 高ティアのモノリスでは100%が必須です。あるいは、「クリティカルからのボーナスダメージ減少」をスタックさせましょう。
3.  **耐久力（Endurance）:** 60%の耐久力キャップを目指し、耐久力閾値（Endurance Threshold）が最大体力の少なくとも20%をカバーするようにします。
4.  **装甲（Armor）:** 筋力（Strength）ビルド以外でも、装甲はすべてのヒット（物理だけでなく）に対して大幅なダメージ軽減を提供します。

## ルートフィルターの書き出しとビルドの共有

**Last Epoch ビルドプランナー**でキャラクターを完成させたら、ルートフィルターを手動で作り直す必要はありません。ほとんどのプランナーには「ルートフィルターのエクスポート（Export Loot Filter）」機能があります。このツールはプランナーで装備したアイテムを分析し、`.xml` ファイルまたはコピー＆ペースト用の文字列を生成します。

この文字列を Last Epoch にインポートすると、ビルドに必要な特定のアフィックスを持つアイテムが自動的に強調表示されます。これは、不要なドロップを排除し、アップグレードに集中できるため、2026年において最も効率的な装備収集方法です。

作成したビルドをコミュニティと共有するには、「Save and Get Link（保存してリンクを取得）」をクリックするだけです。これにより、選択したすべてのノードとアフィックスを保持したユニークなURLが生成され、フォーラムやDiscordでフィードバックを求める際に便利です。

## FAQ

**Q: Last Epoch ビルドプランナーはスマホでも使えますか？**

A: はい、Last Epoch Tools と Maxroll はどちらもモバイルレスポンシブなデザインを採用しています。Maxroll のプランナーは一般的に小さな画面でもスムーズに動作しますが、Last Epoch Tools はベテランプレイヤーが好むより詳細なデータを提供しています。

**Q: プランナーに表示されるダメージがゲーム内より高いのはなぜですか？**

A: プランナーは、クリティカルヒット、状態異常、および条件付きバフ（「体力が満タンの時のダメージ」など）を考慮した「平均DPS」または「実効DPS」を計算することが多いためです。これらはゲーム内のキャラクターシート上ではアクティブになっていない場合があります。プランナーの「Conditions（条件）」タブが、実際のプレイ状況と一致しているか確認してください。

**Q: プランナーは新しいシーズン4のユニークアイテムに対応していますか？**

A: はい、**Last Epoch ビルドプランナー**のデータベースは通常、パッチリリースから数時間以内に更新されます。2026年現在、Shattered Omensのユニークアイテムや新しい「オーメン（Omen）」クラフト素材も、シミュレーションツールに完全に統合されています。

**Q: プランナーで自分の「実効ヘルスプール（EHP）」を確認するにはどうすればよいですか？**

A: 「Defense（防御）」または「Calculations（計算）」タブを探してください。プランナーは体力、装甲、耐性、ワードを組み合わせて、さまざまなダメージタイプ（物理、火炎など）に対するEHP値を算出します。これが生存能力を測る最も正確な方法です。
