## 루트 필터 인터페이스 이해하기

경험을 맞춤 설정하려면 **Shift + F**를 눌러 루트 필터 메뉴를 엽니다. 이 인터페이스에서는 새 필터를 처음부터 만들거나, 기존 필터를 복제하거나, 커뮤니티에서 만든 설정을 가져올 수 있습니다. 2026년 기준으로 시스템은 필터당 최대 200개의 개별 규칙을 지원하므로, 게임 내 모든 아이템 유형과 접사를 세밀하게 관리하기에 충분한 공간을 제공합니다.

### 표시 설정 옵션

규칙을 만들 때 첫 번째 결정은 "표시 설정(Visibility)"입니다. 이는 지정된 조건에 맞는 아이템을 게임에서 어떻게 처리할지 결정합니다.

| 표시 설정 | 기능 | 최적의 사용 사례 |
| :--- | :--- | :--- |
| **표시(Show)** | 아이템 이름표를 정상적으로 표시합니다. | 줍고 싶을 만한 일반적인 아이템. |
| **숨기기(Hide)** | 보기에서 아이템 이름표를 완전히 제거합니다. | 저레벨 베이스나 관련 없는 클래스 필터링. |
| **색상 변경(Recolor)** | 이름표 색상을 변경하고 강조 효과를 추가할 수 있습니다. | **숭고(Exalted)**, **고유(Unique)** 또는 **BIS** 아이템 강조. |

> **💡 팁:** 색상 변경 규칙에서 "강조(Emphasize)" 토글을 사용하면 이름표 텍스트가 더 커지고 대문자로 표시되어, 치열한 전투 중에도 중요한 드롭을 절대 놓치지 않게 됩니다.

## 규칙 우선순위의 논리

가장 중요한 개념은 **라스트 에포크 루트 필터**가 하향식(Top-down) 우선순위 시스템으로 작동한다는 것입니다. 게임은 목록의 맨 위부터 규칙을 대조하며 아이템 드롭을 확인합니다. 아이템이 규칙에 일치하면 해당 규칙의 지침을 따르고 그 아래의 모든 규칙은 무시됩니다.

### 규칙 계층 구조 예시

| 우선순위 | 규칙 유형 | 결과 |
| :--- | :--- | :--- |
| **최상단 (1)** | 모든 **고유(Uniques)** 표시 | 다른 규칙과 상관없이 모든 고유 아이템이 표시됨. |
| **중간 (2)** | 모든 **단검(Daggers)** 숨기기 | 고유 아이템이 아닌 모든 단검이 이제 숨겨짐. |
| **최하단 (3)** | **T7 접사** 표시 | T7 단검은 규칙 2에서 이미 처리되었으므로 숨겨진 상태로 유지됨. |

위의 예를 수정하려면 "T7 접사 표시" 규칙을 "모든 단검 숨기기" 규칙 위로 이동해야 합니다. 이렇게 하면 대부분의 단검은 숨겨지지만, 7티어 접사가 붙은 예외적으로 희귀한 단검은 여전히 화면에 나타납니다.

## 처음부터 유니버설 필터 만들기

2026년 상위 플레이어들 사이에서 인기 있는 전략은 "숨바꼭질(Hide and Seek)" 방식입니다. 이는 기본적으로 거의 모든 것을 숨긴 다음, 높은 우선순위 규칙을 통해 원하는 아이템을 구체적으로 "찾는" 방식입니다.

### 1단계: "전체 숨기기" 규칙
목록의 맨 아래에 규칙을 만듭니다. 표시 설정을 **숨기기**로 설정하고 희귀도를 **일반(Normal)**, **마법(Magic)**, **희귀(Rare)**로 선택합니다. 이렇게 하면 게임 중반으로 넘어갈 때 가치가 낮은 전리품 때문에 방해받지 않습니다.

### 2단계: 클래스 필터링
캠페인 초기에는 모든 클래스의 아이템을 보고 싶을 수 있지만, 레벨 20쯤 되면 플레이하지 않는 클래스의 장비를 숨기는 규칙을 추가해야 합니다. 
*   **조건:** 아이템 유형 -> 특정 하위 유형 (예: 아코라이트 투구, 원시술사 유물).
*   **표시 설정:** 숨기기.
*   **배치:** 전체 숨기기 규칙 위.

### 3단계: 제작용 접사 강조
원하는 접사가 붙은 아이템에 **분쇄의 룬(Rune of Shattering)**을 사용하여 얻을 수 있는 [제작용 파편](https://lastepoch.com/)이 지속적으로 필요합니다.
*   **색상 변경** 규칙을 만듭니다 (일반적으로 주황색이나 청록색).
*   * 필요한 특정 접사를 선택합니다 (예: **복합 생명력**, **스킬 레벨 +1**, **저항**).
*   * 조건을 "이 접사 중 최소 1개 포함"으로 설정합니다.

### 4단계: 우상(Idol) 관리
우상은 제작이 불가능하기 때문에 특별합니다. 올바른 능력치가 붙은 것을 직접 찾아야 합니다.
*   우상 전용 **색상 변경** 규칙을 만듭니다.
*   클래스 전용 접사와 일반적인 "생명력" 또는 "저항" 능력치를 필터링합니다.
*   우상은 희귀하기 때문에 많은 플레이어가 처음에는 모든 우상을 표시하다가 빌드가 완성됨에 따라 필터를 좁히는 방식을 선택합니다.

## 고급 전략: 분쇄, 핑크, 대혼란 규칙

2026년 업데이트 기준으로, 최상위 필터는 아이템이 바닥에 떨어지는 순간 플레이어가 무엇을 해야 할지 신호를 주기 위해 색상 코딩을 자주 사용합니다.

### "절실함" (핑크) 규칙
**투척 마나 소모량**이나 **수리검 레벨**과 같이 제작을 완료하는 데 필요한 특정 파편이 부족한 경우, 필터 맨 위에 해당 접사가 있는 아이템을 **핑크색**으로 변경하는 규칙을 만듭니다. 이는 특정 파편을 확정적으로 얻기 위해 분쇄 대신 **제거의 룬(Rune of Removal)**을 사용해야 한다는 신호입니다.

### 대혼란의 룬(Rune of Havoc) 기법
**대혼란의 룬**은 접사 티어를 바꿀 수 있는 강력한 제작 도구입니다. 만약 7티어(숭고) 아이템이 쓸모없는 주요 능력치와 훌륭한 보조 능력치를 가지고 드롭된다면, 대혼란을 사용하여 해당 T7 수치를 실제로 원하는 능력치로 옮겨볼 수 있습니다.
*   **규칙:** 임의의 T7 접사 + 원하는 접사 1개가 포함된 아이템을 **초록색**으로 변경합니다.
*   **행동:** 이 아이템들을 제작대로 가져가서 "대혼란"을 시도하여 [상인 길드](https://maxroll.gg/last-epoch/resources/merchants-guild-guide)나 개인용으로 쓸 만한 종결급 장비를 만듭니다.

| 색상 | 의미 | 필요한 행동 |
| :--- | :--- | :--- |
| **빨강/금색** | **졸업급(BIS)** | 즉시 교체하거나 전설 잠재력 합성 후보. |
| **주황색** | **분쇄(Shatter)** | 일반 파편을 위해 분쇄의 룬 사용. |
| **핑크색** | **절실함(Dire Need)** | 특정 희귀 파편을 위해 제거의 룬 사용. |
| **초록색** | **승천/대혼란** | 제작 도박을 위해 승천의 룬 또는 대혼란의 룬 사용. |

## 엔드게임 운명의 모노리스 필터링

85레벨 이상에 도달하면 **라스트 에포크 루트 필터** 요구 사항이 바뀝니다. 이 단계에서는 6티어 접사는 종종 "평범한" 것으로 간주되며, 거의 전적으로 7티어 숭고 아이템과 **전설 잠재력(LP)**이 있는 고유 아이템에 집중해야 합니다.

### 티어 요구 사항 설정
접사 조건에서 **고급 옵션(Advanced Options)**을 클릭합니다. 총 접사 티어 합계가 "20 이상"이거나 개별 접사가 "7티어"인 아이템만 표시하도록 필터를 설정할 수 있습니다. 이는 높은 오염도의 모노리스에서 화면 가림을 획기적으로 줄여주어 빠른 클리어 속도를 유지하게 해줍니다.

### 고유 및 세트 아이템 안전 장치
어떤 아이템이 계정에 쓸모없는지 완전히 익숙해지기 전까지는 항상 필터 최상단에 모든 고유 및 세트 아이템을 **표시**하는 규칙을 유지하세요. 2026년에는 "나쁜" 고유 아이템이라도 전설 잠재력이 3~4개 붙어 드롭되면 숭고 아이템과 합성하여 전설 아이템을 만들 수 있으므로 가치가 있습니다.

> **⚠️ 경고:** 해당 아이템들을 정말로 놓치고 싶은 것이 아니라면 "고유 표시" 규칙 위에 "숨기기" 규칙을 절대 두지 마세요. 규칙 우선순위는 절대적입니다!

## FAQ

**Q: 하나의 라스트 에포크 루트 필터를 모든 캐릭터에 사용할 수 있나요?**

A: 네, "유니버설" 필터를 만들 수 있습니다. 이를 위해 "클래스 요구 사항" 조건을 사용하여 현재 클래스의 아이템은 표시하고 다른 클래스는 숨기세요. 하지만 엔드게임에 도달하면 특정 BIS 접사를 고려하여 각 빌드에 특화된 필터를 사용하는 것이 일반적으로 더 좋습니다.

**Q: 온라인에서 찾은 필터는 어떻게 가져오나요?**

A: 필터 텍스트나 XML 내용을 클립보드에 복사합니다. 게임 내 루트 필터 인터페이스(Shift+F)를 열고 **+** 버튼을 클릭한 다음 "클립보드 내용 붙여넣기(Paste Clipboard Contents)"를 선택합니다. 필터가 미리 정의된 모든 규칙으로 자동으로 채워집니다.

**Q: "숨기기" 규칙이 있는데도 왜 일부 아이템이 여전히 보이나요?**

A: 규칙 우선순위를 확인하세요. 아이템이 먼저 일치하는 "표시" 또는 "색상 변경" 규칙이 "숨기기" 규칙보다 위에 있을 가능성이 높습니다. 게임은 위에서부터 첫 번째 일치하는 규칙을 찾는 즉시 확인을 중단한다는 점을 기억하세요.

**Q: 전설 잠재력을 필터링하는 가장 좋은 방법은 무엇인가요?**

A: **희귀도: 고유** 조건을 가진 규칙을 추가한 다음, **고급 옵션**을 사용하여 "전설 잠재력 1 이상"을 지정하세요. 1LP, 2LP, 3LP 아이템에 서로 다른 색상을 부여하는 여러 규칙을 만들어 시각적 명확성을 높일 수 있습니다.
